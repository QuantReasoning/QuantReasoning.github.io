About Me

Biomedical researcher and aspiring data scientist with a First Class BSc in Medical Sciences (University of Exeter) and currently enrolled at MIT in Statistics and Data Science. I combine a strong foundation in medical and molecular biology with computational expertise in Python and R, applying machine learning, deep learning, single-cell RNA-seq, and multi-omics integration to translational research.

I built QuantReasoning as a portfolio website to showcase my coding projects, ranging from statistical modelling to bioinformatics pipelines, and to deepen my programming skills and passion for applied statistics and computational biology. Experienced in bridging wet lab, clinical, and data science domains to support biomarker discovery and precision medicine.

Selected Projects

Single-Cell RNA-seq Analysis
This project performs single-cell transcriptomic analysis of human lung cancer FFPE tissue using R (Seurat, DESeq2, ggplot2) on publicly available 10x Genomics data to identify and visualize cell types and differentially expressed genes.
QSAR SARS-CoV
QSAR model using Random Forest regression to predict molecular bioactivity (pIC₅₀) from ECFP4 fingerprints. The workflow includes data preprocessing, stratified splitting, and feature importance analysis to highlight key substructures influencing activity.
Streamlit Statistical Web App
Statpy is a no-code statistical web app that helps users run t-tests, ANOVA, Chi-square, and correlation analyses directly in the browser. Designed for accessibility, it empowers researchers, students, and analysts to explore their data and interpret results without writing a single line of code.
Patient Risk Stratification (R)
Created a synthetic clinical dataset and implemented a multinomial logistic regression model in R to stratify patients into Low, Medium, and High risk groups based on age, diagnosis, and length of stay; The model achieved an overall accuracy of 88.3%.
KNN and Cross Validation Cancer Model
This project uses Python and the K-Nearest Neighbors (KNN) algorithm to classify tumors in the Breast Cancer Wisconsin dataset, achieving a test set accuracy of 95.7% through feature scaling, cross-validation, and evaluation with precision, recall, F1-score, and a confusion matrix.
Diabetes Prediction Model (R)
Developed a logistic regression model in R to predict diabetes risk using the Pima Indians dataset, with 76% accuracy.
Cancer Cell Classification
A supervised classification analysis in Python, on data derived from breast cancer cell nuclei features, using pandas, scikit-learn, matplotlib, and seaborn to classify tumors as malignant or benign, achieving 94–96% accuracy with a Naive Bayes classifier.
